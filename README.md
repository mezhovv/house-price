# House Price Prediction

Небольшой проект по предсказанию стоимости недвижимости с использованием классического машинного обучения.

Цель - построить модель, которая на основе характеристик района и самого дома (количество комнат, уровень преступности, удаленность от центров и т.д.) предсказывает его рыночную стоимость.

Модель была обучена и протестирована на основе датасета boston house prices с kaggle.

## Стек технологий
* **Язык:** Python 3.10+
* **Библиотеки:**
    *  `pandas`, `numpy` — для обработки данных.
    * `matplotlib`, `seaborn` — для визуализации и анализа (EDA).
    * `scikit-learn` — для построения модели и предобработки.

## Результаты
MAE для линейной регрессии: 3.15

MAE для линейной регрессии с масштабированием значений (линейная регрессия лучше работает со значениями одинакового масштаба): 3.15

MAE для случайного леса: 2.06

## Как запустить проект локально
Установить зависимости.
